Kako dodati CSS u HTML

U ovoj lekciji naučili smo različite načine dodavanja CSS-a u HTML dokument:

  • Inline CSS direktno u tagu
  • Unutarnji CSS unutar <style> taga u <head>
  • Vanjski CSS povezivanjem vanjske .css datoteke
<p style="color:red">Tekst</p>

<style>
  p { color:red; }
</style>

<link rel="stylesheet" href="style.css">

CSS Box Model

CSS Box Model opisuje kako su elementi prikazani kao pravokutnici na stranici.

  • Sadržaj (content)
  • Padding (unutarnji razmak)
  • Border (okvir)
  • Margin (vanjski razmak)
.box {
  width: 200px;
  padding: 20px;
  border: 3px solid black;
  margin: 15px;
}

CSS Box i Background

CSS background omogućava postavljanje boje ili slike pozadine pomoću background-color i background-image.

div {
  background-color: lightblue;
}

Opacity

Opacity određuje prozirnost elementa (0 = potpuno prozirno, 1 = potpuno vidljivo).

.kutija {
  opacity: 0.5;
}

CSS Pozicioniranje

Vrijednosti position svojstva su: static, relative, absolute, fixed i sticky.

.kutija {
  position: relative;
  top:20px;
}
